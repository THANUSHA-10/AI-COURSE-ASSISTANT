# AI Course Assistant Chatbot

### 1. Problem Statement & User Persona
* **Problem:** Students lose track of grading breakdowns, assignment deadlines, and course policies scattered across complex documents.
* **Persona ("Overwhelmed Ollie"):** A busy student who needs fast, direct clarification on course policies late at night without waiting for an email reply.

### 2. Functional Prototype
This app uses Python, Streamlit, and the OpenAI `gpt-4o-mini` engine to create a sandboxed conversational interface acting as a course teaching assistant.

### 3. Reflection
* **Strength:** It gives instant, accurate answers 24/7 without taking up the instructor's manual time.
* **Limitation:** It requires an external OpenAI API key to run and relies completely on the rules set in the system prompt to avoid making up answers.
